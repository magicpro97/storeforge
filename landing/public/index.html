🚀 App Store & Google Play CLI

Deploy to App Store & Google Play from your terminal

One CLI to upload builds, sync metadata, check review status, and release to production on both iOS and Android.

$ npm install -g storeforge
Terminal
$ storeforge upload ios ./build/MyApp.ipa
✔ IPA uploaded successfully (42.8 MB) to TestFlight

$ storeforge upload android ./build/app-release.aab
✔ AAB uploaded to internal track (38.2 MB)

$ storeforge status
📱 App Store Connect
  Status: 🔍 In Review — v2.1.0
🤖 Google Play
  Status: ✅ Live — v210

$ storeforge release android production
✔ Build promoted to production!

Everything you need to ship

One CLI for both stores. No more juggling dashboards.

📤

Upload Builds

Upload IPA to TestFlight and AAB to Google Play with a single command. Supports internal, alpha, beta, and production tracks.

📝

Metadata Sync

Keep app metadata in version-controlled YAML files. Sync titles, descriptions, keywords, and screenshots to both stores.

🔍

Review Status

Check review and processing status on both stores at once. Know exactly where your builds are in the pipeline.

🚀

Release Management

Submit iOS builds for review and promote Android builds through tracks — from internal all the way to production.

🔐

Secure Config

API keys and service accounts stored securely in your home directory. Interactive setup wizard gets you going in minutes.

🏪

Multi-Store

Unified interface for App Store Connect and Google Play. Same workflows, same commands — for both platforms.

Get started in 3 steps

From install to first deploy in under 5 minutes.

1

Install StoreForge

npm install -g storeforge
2

Configure credentials

Run the interactive setup wizard to connect your App Store Connect API key and Google Play service account.

storeforge init
3

Upload & release

Upload your build and promote to production when ready.

storeforge upload ios ./MyApp.ipa
